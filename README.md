## Install

1. **Clone Repository**

```bash
git clone https://github.com/adhiariyadi/Staycation-Backend-ExpressJS.git
cd Staycation-Backend-ExpressJS
```

2. **Instalasi website**

```bash
npm install
node seeder.js
```

3. **Jalankan website**

```bash
npm run dev
```
